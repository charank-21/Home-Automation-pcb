# Home Automation PCB

Schematic capture and PCB layout design of a home automation system using Cadence Allegro. This project includes schematic design, PCB development, Gerber generation, BOM preparation, and 3D visualization.

## Tools Used
- Cadence Allegro

## Project Overview
This project focuses on the design and development of a PCB for a home automation system. The design process includes schematic capture, component selection, PCB layout creation, design verification, and manufacturing file generation.

## Project Files
- Schematic Diagram
- Gerber Output
- Bill of Materials (BOM)
- 3D PCB View
- Design Files (ZIP)

## Design Flow
1. Schematic Capture
2. Component Selection
3. PCB Layout Design
4. Design Rule Check (DRC)
5. Gerber Generation
6. BOM Preparation
7. 3D Verification
